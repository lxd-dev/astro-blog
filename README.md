# Astro Blog

A personal developer blog built with Astro.

This project is used to publish my learning notes on programming, tools, and software development.

## Tech Stack

- Astro
- Markdown
- Cloudflare Pages
- GitHub

## Local Development

Install dependencies:

npm install

Start the dev server:

npm run dev

Open:

http://localhost:4321

## Deployment

The site is automatically deployed via Cloudflare Pages whenever changes are pushed to the main branch.
